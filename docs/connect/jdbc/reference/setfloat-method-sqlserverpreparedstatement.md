# <a name="setfloat-method-sqlserverpreparedstatement"></a>SetFloat-Methode (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Legt den angegebenen Parameter auf den angegebenen **"float"** Wert.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public final void setFloat(int n,  
                           float x)  
```  
  
#### <a name="parameters"></a>Parameter  
 *n*  
  
 Ein **Int** , der die Parameteranzahl angibt.  
  
 *x*  
  
 Ein **"float"** Wert.  
  
## <a name="exceptions"></a>Ausnahmen  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Hinweise  
 Diese SetFloat-Methode wird von der SetFloat-Methode in der java.sql.PreparedStatement-Schnittstelle angegeben.
