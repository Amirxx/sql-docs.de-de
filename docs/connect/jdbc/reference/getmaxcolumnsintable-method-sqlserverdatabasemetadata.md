# <a name="getmaxcolumnsintable-method-sqlserverdatabasemetadata"></a>getMaxColumnsInTable-Methode (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Ruft die maximale Anzahl von Spalten, die dieser Datenbank in einer Tabelle zulässig.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public int getMaxColumnsInTable()  
```  
  
## <a name="return-value"></a>Rückgabewert  
 Ein **Int** , der die maximale Anzahl zulässiger Spalten angibt.  
  
## <a name="exceptions"></a>Ausnahmen  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Hinweise  
 Diese GetMaxColumnsInTable-Methode wird von der GetMaxColumnsInTable-Methode in der java.sql.DatabaseMetaData-Schnittstelle angegeben.
