# Editor f&#252;r den Task Anmeldungen &#252;bertragen (Seite Allgemein)
  Mithilfe der Seite **Allgemein** des Dialogfelds **Editor für den Task Anmeldungen übertragen** können Sie den Task Fehlermeldungen übertragen benennen und beschreiben. Weitere Informationen zu diesem Task finden Sie unter [Transfer Logins Task](../../integration-services/control-flow/transfer-logins-task.md).  
  
## enthalten  
 **Name**  
 Geben Sie für den Task Anmeldungen übertragen einen eindeutigen Namen ein. Dieser Name wird im Tasksymbol als Bezeichnung verwendet.  
  
> [!NOTE]  
>  Tasknamen müssen innerhalb eines Pakets eindeutig sein.  
  
 **Description**  
 Geben Sie eine Beschreibung des Tasks Anmeldungen übertragen ein.
