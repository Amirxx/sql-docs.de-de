# <a name="azure-hdinsight-connection-manager"></a>Azure HDInsight-Verbindungs-Manager
Die **Azure HDInsight-Verbindungs-Manager** ermöglicht einem SSIS-Paket für die Verbindung mit einem Azure HDInsight-Cluster.

Die **Azure HDInsight-Verbindungs-Manager** ist eine Komponente von der [SQL Server Integration Services (SSIS) Feature Pack für Azure](../../integration-services/azure-feature-pack-for-integration-services-ssis.md).

Zum Erstellen und Konfigurieren einer **Azure HDInsight-Verbindungs-Manager**, befolgen Sie die folgenden Schritte aus:

1. In der **SSIS-Verbindungs-Manager hinzufügen** wählen Sie im Dialogfeld **AzureHDInsight**, und klicken Sie auf **hinzufügen**.
2. In der **Azure HDInsight-Verbindungs-Manager-Editor** Dialogfeld geben die **Cluster-DNS-Namen** (ohne das Protokollpräfix) **Benutzername**, und **Kennwort** für den HDInsight-Cluster für die Verbindung.
3. Klicken Sie auf **OK** , um das Dialogfeld zu schließen.
4. Die Eigenschaften des Verbindungs-Managers, die Sie im Fenster **Eigenschaften** erstellt haben, werden angezeigt.
