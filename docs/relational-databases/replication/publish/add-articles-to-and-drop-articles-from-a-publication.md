# Hinzuf&#252;gen von Artikeln zu einer Ver&#246;ffentlichung und L&#246;schen von Artikeln aus einer Ver&#246;ffentlichung (SQL Server Management Studio)
  Sie können einer Veröffentlichung bereits bei Ihrer Erstellung im Assistenten für neue Veröffentlichung Artikel hinzufügen. Weitere Informationen zum Verwenden dieses Assistenten finden Sie unter [Erstellen einer Publikation](../../../relational-databases/replication/publish/create-a-publication.md).  
  
 Nachdem eine Veröffentlichung erstellt wurde, hinzufügen und Löschen von Artikeln zu den **Artikel** auf der Seite der **Veröffentlichungseigenschaften - \< Veröffentlichung>** (Dialogfeld). Weitere Informationen zum Zugreifen auf dieses Dialogfeld finden Sie unter [View and Modify Publication Properties](../../../relational-databases/replication/publish/view-and-modify-publication-properties.md). Informationen zu Überlegungen für das Hinzufügen und Löschen von Artikeln finden Sie unter [Hinzufügen und löschen Artikeln aus vorhandenen Veröffentlichungen](../../../relational-databases/replication/publish/add-articles-to-and-drop-articles-from-existing-publications.md).  
  
### So fügen Sie einer bereits erstellten Veröffentlichung einen Artikel hinzu  
  
1.  Auf der **Artikel** auf der Seite der **Veröffentlichungseigenschaften - \< Veröffentlichung>** Deaktivieren Sie im Dialogfeld die **nur aktivierte Objekte in der Liste anzeigen** das Kontrollkästchen. Auf diese Weise werden nur die nicht veröffentlichten Objekte in der Veröffentlichungsdatenbank angezeigt.  
  
2.  Markieren Sie die Kontrollkästchen für alle Artikel, die Sie der Veröffentlichung hinzufügen möchten.  
  
3.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]  
  
### So löschen Sie einen Artikel  
  
1.  Auf der **Artikel** auf der Seite der **Veröffentlichungseigenschaften - \< Veröffentlichung>** im Dialogfeld deaktivieren das Kontrollkästchen neben jeder Artikel, die Sie löschen möchten.  
  
2.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]
