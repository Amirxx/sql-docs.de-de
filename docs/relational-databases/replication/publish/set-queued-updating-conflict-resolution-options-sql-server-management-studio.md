# Festlegen der Konfliktl&#246;sungsoptionen f&#252;r verz&#246;gerte Updates &#252;ber eine Warteschlange (SQL Server Management Studio)
  Festgelegt Konfliktlösungsoptionen für Veröffentlichungen, die Unterstützung für Abonnements mit Aktualisierung verzögerter auf die **Abonnementoptionen** auf der Seite der **Veröffentlichungseigenschaften - \< Veröffentlichung>** (Dialogfeld). Weitere Informationen zum Zugreifen auf dieses Dialogfeld finden Sie unter [View and Modify Publication Properties](../../../relational-databases/replication/publish/view-and-modify-publication-properties.md).  
  
### So legen Sie die Konfliktlösungsoptionen für das verzögerte Update über eine Warteschlange fest  
  
1.  Auf der **Abonnementoptionen** auf der Seite der **Veröffentlichungseigenschaften - \< Veröffentlichung>** Wählen Sie im Dialogfeld einen der folgenden für Werte die **Konflikt Richtlinie** Option:  
  
    -   **Verlegeränderung beibehalten**  
  
    -   **Abonnentenänderung beibehalten**  
  
    -   **Abonnement erneut initialisieren**  
  
2.  [!INCLUDE[clickOK](../../../includes/clickok-md.md)]
