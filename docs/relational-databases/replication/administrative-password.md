# Administratorkennwort
  Wenn ist, auf die **Distributoren** Seite dieses Assistenten einen Remoteverteiler für diesen Verleger ausgewählten eingeben, müssen Sie ein Kennwort für die Replikation Verbindung zwischen dem Verleger und dem Verteiler mithilfe der **Distributor_admin** Anmeldung. Das Kennwort muss dem Kennwort entsprechen, das auf der Seite **Verteilerkennwort** des Verteilungskonfigurations-Assistenten oder auf der Seite **Verleger** des Dialogfelds **Verteilereigenschaften** angegeben wurde.  
  
## Optionen  
 **Kennwort**  
 Geben Sie das Kennwort für die Verbindung zwischen dem Verleger und dem Remoteverteiler ein.  
  
 **Kennwort bestätigen**  
 Geben Sie das Kennwort zur Bestätigung der fehlerfreien Eingabe erneut ein.
