# MSSQL_ENG020575
    
## Meldungsdetails  
  
|||  
|-|-|  
|Produktname|SQL Server|  
|Ereignis-ID|20575|  
|Ereignisquelle|MSSQLSERVER|  
|Komponente|[!INCLUDE[ssDEnoversion](../../includes/ssdenoversion-md.md)]|  
|Symbolischer Name||  
|Meldungstext|Die Datenüberprüfung für das Abonnement des Abonnenten '%s' für den '%s'-Artikel in der '%s'-Veröffentlichung wurde erfolgreich durchlaufen.|  
  
## Erklärung  
 Die Daten auf dem Abonnenten wurden gegen die Daten auf dem Verleger abgeglichen, und es wurde eine Übereinstimmung der Daten festgestellt. Daher wurde die Überprüfung erfolgreich durchlaufen. Weitere Informationen zur Überprüfung finden Sie unter [Überprüfen von replizierten Daten](../../relational-databases/replication/validate-replicated-data.md).
