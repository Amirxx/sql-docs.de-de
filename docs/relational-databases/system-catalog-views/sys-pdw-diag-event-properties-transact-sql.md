# <a name="syspdwdiageventproperties-transact-sql"></a>Sys.pdw_diag_event_properties (Transact-SQL)
[!INCLUDE[tsql-appliesto-xxxxxx-xxxx-xxxx-pdw-md](../../includes/tsql-appliesto-xxxxxx-xxxx-xxxx-pdw-md.md)]

  Enthält Informationen darüber, welche Eigenschaften Diagnoseereignisse zugeordnet sind.  
  
|Spaltenname|Datentyp|Description|Bereich|  
|-----------------|---------------|-----------------|-----------|  
|**Ereignisname**|**nvarchar(255)**|Der Name des Ereignisses bestimmte Diagnosefunktionen.||  
|**Eigenschaftsname**|**nvarchar(255)**|Der Name einer Eigenschaft des Ereignisses.||
