# <a name="flagname-property-serversettingsgeneralflag-class"></a>FlagName-Eigenschaft (ServerSettingsGeneralFlag-Klasse)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Ruft den Namen des Flags ab.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
object.FlagName [= value]  
```  
  
## <a name="parts"></a>Bestandteile  
 *Objekt*  
 Ein [ServerSettingsGeneralFlag-Klassenobjekt](../../../relational-databases/wmi-provider-configuration-classes/serversettingsgeneralflag-class/serversettingsgeneralflag-class.md) , das ein allgemeines Flag für die Servereinstellungen darstellt.  
  
## <a name="property-valuereturn-value"></a>Eigenschaftswert/Rückgabewert  
 Ein Zeichenfolgenwert, der den Namen des Flags angibt.
