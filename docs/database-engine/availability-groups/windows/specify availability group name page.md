# Seite Namen der Verf&#252;gbarkeitsgruppe angeben (Assistent f&#252;r neue Verf&#252;gbarkeitsgruppen/Assistent zum Hinzuf&#252;gen von Datenbanken)
  In diesem Thema werden die Optionen der Seite **Namen der Verfügbarkeitsgruppe angeben** beschrieben. Dieses Thema wird sowohl von [!INCLUDE[ssAoNewAgWiz](../../../includes/ssaonewagwiz-md.md)] als auch von [!INCLUDE[ssAoAddDbWiz](../../../includes/ssaoadddbwiz-md.md)] von [!INCLUDE[ssCurrent](../../../includes/sscurrent-md.md)] verwendet.  
  
##  <a name="PageOptions"></a> Optionen für "Namen der Verfügbarkeitsgruppe angeben"  
 **Namen einer Verfügbarkeitsgruppe angeben**  
 Geben Sie den Namen der Verfügbarkeitsgruppe an. Geben Sie für eine neue Verfügbarkeitsgruppe einen gültigen [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]-Bezeichner an, der in allen Verfügbarkeitsgruppen im WSFC-Cluster eindeutig ist. Die maximale Länge eines Verfügbarkeitsgruppennamens beträgt 128 Zeichen.
