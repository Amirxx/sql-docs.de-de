# Vorbereiten der Aktualisierung durch Ausf&#252;hren von Upgrade Advisor
  Bereiten Sie die Aktualisierung Ihrer OLTP-Datenbanken und die Übernahme neuer Funktionen in [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] vor, indem Sie Microsoft Data Migration Assistant herunterladen und ausführen.  
  
## <a name="download-and-install-data-migration-assistant"></a>Herunterladen und Installieren von Data Migration Assistant  
 Laden Sie Data Migration Assistant [hier](http://go.microsoft.com/fwlink/?LinkID=613421) herunter, und installieren Sie ihn. Dieses Tool ist in den [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] -Installationsmedien nicht enthalten.
