# <a name="discontinued-functionality-to-sql-server-reporting-services-in-sql-server"></a>Nicht mehr unterstützte Funktionen in SQL Server Reporting Services in SQL Server 

  In diesem Thema wird beschrieben, [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] Funktionen, die in SQL Server 2016 ohne mehr verfügbar sind. Das Thema enthält keine Hinweise zu Versionen von Betriebssystemen oder [!INCLUDE[msCoName](../includes/msconame-md.md)] Internet Information Services (IIS), die nicht mehr unterstützt werden. Weitere Informationen zu den Systemanforderungen finden Sie unter [Hardware- und Softwareanforderungen für die Installation von SQL Server 2016](../sql-server/install/hardware-and-software-requirements-for-installing-sql-server.md).  
  
Eine *nicht mehr unterstützte Funktion* ist eine veraltete Funktion, die nicht länger unterstützt wird. Eventuell wurde sie sogar aus dem Produkt entfernt. Die folgenden Funktionen werden nicht mehr unterstützt.  
  
|Funktion|Alternative oder Problemumgehung|
|-|-|
|Hochladen von Berichtsmodellen über [!INCLUDE[ssRSWebPortal](../includes/ssrswebportal.md)]|Dies kann weiterhin über die SOAP-API erfolgen.|  
|Verwalten von Berichtsmodellen über [!INCLUDE[ssRSWebPortal](../includes/ssrswebportal.md)]|Dies kann weiterhin über die SOAP-API erfolgen.|
