# SendUsingSMTPServer-Eigenschaft (WMI: MSReportServer_ConfigurationSetting)
  Gibt an, ob das **SendUsing**-Element in der E-Mail-Konfiguration für den Berichtsserver auf **true** festgelegt wurde.  
  
## Syntax  
  
```vb  
Public Dim SendUsingSMTPServer As Boolean  
```  
  
```csharp  
public Boolean SendUsingSMTPServer;  
```  
  
## Eigenschaftswerte  
 Ein schreibgeschütztes **boolesches** Objekt.
