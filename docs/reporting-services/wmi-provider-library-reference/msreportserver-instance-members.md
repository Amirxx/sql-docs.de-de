# MSReportServer_Instance-Member
  Die MSReportServer_Instance-Klasse stellt grundlegende Informationen bereit, die ein Client benötigt, um eine Verbindung mit einem installierten Berichtsserver herzustellen.  
  
## Öffentliche Eigenschaften  
  
|||  
|-|-|  
|[EditionID](../../reporting-services/wmi-provider-library-reference/editionid-property-wmi-msreportserver-instance.md)|Gibt die Edition des Berichtsservers zurück.|  
|[EditionName](../../reporting-services/wmi-provider-library-reference/editionname-property-wmi-msreportserver-instance.md)|Gibt den lesbaren Namen der Edition des Berichtsservers zurück|  
|[InstanceID](../../reporting-services/wmi-provider-library-reference/instanceid-property-wmi-msreportserver-instance.md)|Dieses Objekt unterstützt die [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] -Infrastruktur und ist nicht für die direkte Verwendung im Code vorgesehen.|  
|[InstanceName](../../reporting-services/wmi-provider-library-reference/instancename-property-wmi-msreportserver-instance.md)|Gibt den Namen einer Berichtsserverinstanz auf einem bestimmten Computer zurück. Schreibgeschützt.|  
|[IsSharePointIntegrated](../../reporting-services/wmi-provider-library-reference/issharepointintegrated-property-wmi-msreportserver-instance.md)|Gibt an, ob der Berichtsserver für den integrierten SharePoint-Modus konfiguriert ist|  
|[Version](../../reporting-services/wmi-provider-library-reference/version-property-wmi-msreportserver-instance.md)|Gibt die Version des Berichtsservers zurück|  
  
## Öffentliche Methoden
